Single Chrono/Timer screen

Here you can control a single Chrono or Timer (CT).
Icons and Time display act like buttons and are highlighted according to what is activated or no.
To edit the default time and message, press the Time display.
Exit this screen to return to Main screen.

Icon Function
Keep Screen (ON/OFF)
Start/Stop (*) CT
Split CT
Set Alarm in Clock app for CT (if running timer) (ON/OFF)
(shows Alarms screen in Clock app on dismissing)
Reset (*)
Select Chrono mode
(only if timer stopped)
Select Timer mode
(only if chrono stopped)
Time Set & Presets (Default Time/Message)
(calls Presets screen)
(won't validate if running timer with alarm set in Clock app)

(*) If running timer with alarm: shows Alarms screen in Clock app and dismisses alarm

Menu item Function

(and calls Color screen to set its Light On, Light off or Back color)
